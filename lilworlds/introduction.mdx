# Welcome to LilWorlds

**LilWorlds** is a performant, modular world management plugin for Minecraft servers. Designed as a modern alternative to Multiverse-Core, LilWorlds focuses on performance, organization, and modularity while providing all the features you need for advanced world management.

## Quick Command Overview

LilWorlds provides intuitive commands for all world management tasks:

### World Management Commands (`/world` or `/w`)

```bash
# Create worlds
/world create myworld                    # Basic world creation
/world create void_world NORMAL -g VoidGenerator  # With custom generator

# Manage worlds
/world list                             # List all worlds
/world info myworld                     # Detailed world information
/world load myworld                     # Load a world
/world unload myworld                   # Unload a world

# Clone and import
/world clone source target              # Clone existing world
/world import existing_world            # Import external world

# Safe deletion with configuration purging
/world remove myworld                   # Delete with confirmation
```

### Plugin Administration (`/worlds`)

```bash
/worlds reload                          # Reload all configurations
/worlds reload generators               # Reload custom generators
/worlds cleanup --orphaned              # Clean up orphaned files
```
