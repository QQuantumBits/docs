## Plugin Compatibility

<Info>
**Minecraft Versions:** 1.16 - 1.21.5

**Server Software:** Spigot, Paper, Purpur (any Spigot-based)

**Java Version:** 8+ (Java 17+ recommended for optimal performance)

**Dependencies:** None required, optional integrations available
</Info>

## Quick Start

Get started with any of our plugins in minutes:

### 1. Download & Install
Download the latest version from GitHub releases and place in your `plugins/` folder.

### 2. Configure
Each plugin comes with sensible defaults but can be fully customized via YAML configuration files.

### 3. Use Commands
All plugins feature intuitive command systems with tab completion and comprehensive help.

### 4. Explore Features
Check out the detailed documentation for each plugin to unlock advanced features.
