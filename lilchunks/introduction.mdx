## Quick Overview

LilChunks provides a simple yet powerful interface for chunk pregeneration:

```bash
# Start generation with 10 chunk radius
/lilchunks start 10

# Check generation status
/lilchunks status

# Stop generation
/lilchunks stop

# Reload configuration
/lilchunks reload
```

<Tip>
Use the aliases `/lc` or `/chunks` for faster command execution!
</Tip>
